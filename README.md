## Server Side

### Requirements
- Node.js
- npm or yarn

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/WhitelightningDev/client.git
   ```
2. Navigate to the server directory:
   ```bash
   cd server
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

### Running the Server
1. Start the server:
   ```bash
   npm start
   ```
   or
   ```bash
   yarn start
   ```
